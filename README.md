# Lamp Login UI

## Project Description

Lamp Login UI is a simple and creative login interface where a lamp controls the visibility of the login form. When the user clicks on the lamp, the light turns on and reveals the login card with a smooth animation. When the lamp is turned off, the login form disappears.

This project demonstrates basic frontend development concepts such as UI design, animations, and interactive elements using HTML, CSS, and JavaScript.

## Features

* Interactive lamp toggle
* Animated login form
* Modern UI design
* Simple and responsive layout
* Beginner-friendly frontend project

## Technologies Used

* HTML5
* CSS3
* JavaScript
